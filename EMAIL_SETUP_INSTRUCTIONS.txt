# Email Configuration Instructions

## Step 1: Gmail App Password Setup
1. Go to: https://myaccount.google.com/
2. Go to Security → 2-Step Verification (enable it first)
3. Go to Security → App Passwords
4. Select "Mail" and "Other (Custom name)"
5. Name: "Mithun Construction"
6. Copy the 16-character password

## Step 2: Create .env file in backend folder
Create a file named `.env` in the backend folder with:
```
EMAIL_PASSWORD=your_16_character_app_password_here
PORT=5000
NODE_ENV=development
```

## Step 3: Start the backend server
```bash
cd backend
npm start
```

## Step 4: Update frontend to use backend
In frontend/src/components/Contact.jsx, update the form submission to call your backend API.

Your contact form will now send emails to mithungpk@gmail.com!
